# Manufacturing Analysis Dashboard (Power BI)

## 📊 Project Overview

The **Manufacturing Analysis Dashboard** is a Power BI project designed to analyze and monitor manufacturing performance through interactive and visual reporting. The dashboard transforms raw manufacturing data into meaningful insights that help stakeholders track efficiency, quality, production trends, and operational bottlenecks.

This project demonstrates practical skills in **data cleaning, modeling, DAX calculations, and dashboard design** using Power BI.

---

## 🎯 Objectives

* Monitor overall manufacturing performance
* Analyze production output and trends
* Identify defect rates and quality issues
* Track downtime and operational efficiency
* Support data-driven decision-making

---

## 🛠 Tools & Technologies Used

* **Power BI Desktop** – Dashboard development and visualization
* **Power Query** – Data cleaning and transformation
* **DAX (Data Analysis Expressions)** – KPI and measure calculations
* **Microsoft Excel / CSV** – Source data format

---

## 📁 Dataset Description

The dataset represents manufacturing operational data and includes:

* Production date and shift
* Plant / factory details
* Product categories
* Units produced
* Defective units
* Downtime hours
* Production cost

*(Note: Dataset is either simulated or anonymized for academic/portfolio use.)*

---

## 🔄 Data Preparation

* Removed null and duplicate values
* Standardized date and numeric formats
* Created calculated columns and measures
* Established relationships between tables (if applicable)

---

## 📐 Key KPIs & Metrics

* **Total Production Output**
* **Defect Rate (%)**
* **Downtime (Hours)**
* **Production Efficiency**
* **Overall Equipment Effectiveness (OEE)**
* **Cost per Unit**

## 📈 Dashboard Features

* KPI cards for quick performance overview
* Line charts for production trends over time
* Bar/column charts for product and plant comparison
* Slicers for date, product, and plant-level filtering
* Interactive visuals for detailed drill-down analysis

---

## 🔍 Insights Generated

* Identification of high-defect products
* Detection of peak downtime periods
* Comparison of efficiency across plants
* Analysis of production trends over time
* Support for operational improvement strategies

## 🚀 How to Use the Dashboard

1. Download the `.pbix` file from the repository
2. Open it using **Power BI Desktop**
3. Use slicers to filter by date, product, or plant
4. Interact with visuals to explore insights
